# FunkFit Builder v0.4.4 — clean build

Dette er en komplet, selvstændig pakke. Slet alle eksisterende filer i repositoryet (undtagen `.git`) og kopier hele indholdet af denne mappe ind.

## Kontrol

- Orange toptekst skal vise `VERSION 0.4.4`.
- Dropdownen skal vise FunkFit Junior, HIIT og Hyrox-skabeloner.
- Der skal vises sektioner straks ved opstart.
- `+ Tilføj øvelse` skal åbne en øvelsesvælger i den valgte sektion.
- Favoritter, detaljer, instruktionsfilm, musiklinks, fuldskærmsafspiller og PDF-print er inkluderet.

## Ved gammel cache

Åbn `reset-cache.html` én gang på telefonen. Den afregistrerer gamle service workers, sletter cache og åbner v0.4.4 igen.
